Italian Bible Study — native iOS app

Screen-by-screen design of the shipped SwiftUI app in ios-native/. Every frame is labelled with the Swift file that implements it, so this page is both the design source and a map for learning the codebase.

Using this in Figma: install the free html.to.design plugin in Figma → “Import URL / paste HTML” → paste this file’s contents (or serve it and paste the URL). Each phone frame lands as an editable Figma frame. Alternatively: open this file in a browser, screenshot each 375×780 frame (⌘⇧4 on macOS), and drop the PNGs onto a Figma page — 375 pt equals the iPhone 15/16 logical width, so frames align 1:1 with Figma’s iPhone presets. The Style guide below carries the tokens to recreate components natively in Figma.

Components

List cells: insetGrouped, 12 pt corner radius, 0.5 pt separators · Capsule badges: accent @ 13% fill, 11 pt bold (NOW = solid accent, REVIEW = orange) · Primary button: borderedProminent (accent fill, white label) · Secondary: bordered (gray 12% fill) · Tappable words: accent text (vocab) / plain text (other) — every Italian word opens the gloss sheet · Sheets: medium/short detents with grabber · Haptics: success on correct/done, error on wrong, light on flip.
